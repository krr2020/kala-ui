### 1.0.0 - 2025-01-06

#### Added
- Initial release of Kala UI Design Tokens
- Comprehensive design system foundation

#### Tokens Included
- Colors (primary, secondary, neutral, semantic)
- Spacing scale (0-96)
- Typography (font families, sizes, weights, line heights)
- Shadows (box shadows, drop shadows, elevation levels)
- Breakpoints (responsive breakpoints)
- Transitions (durations, timing functions, presets, keyframes)

#### Features
- Full TypeScript type support
- Tailwind CSS integration
- Platform-agnostic design tokens
- Semantic color naming
- Responsive breakpoint definitions
- Animation and transition presets

#### Usage
- Can be imported as complete package or individual token categories
- Integrates seamlessly with Tailwind CSS configuration
- Supports direct usage in styled components
